# next-profile

Plugin Wordpress para adicionar um profile extra e utilizar get_the_author_meta para recuperar um substatus especifico a sua necessidade.

```php
<?php 
    get_the_author_meta( 'extra_profile', $profileuser->ID );
?>
```
| Software  | Version |
|-----------|---------|
| Wordpress | 5.2     |
| PHP       | 7.2     |
